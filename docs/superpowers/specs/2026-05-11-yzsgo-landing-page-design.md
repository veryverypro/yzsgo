# 鸭嘴售（yzsgo.com）国内落地页设计文档

> 日期：2026-05-11

## 概述

为 Optima AI 创建面向国内跨境电商卖家的独立品牌落地页。品牌名"鸭嘴售"（谐音鸭嘴兽），域名 yzsgo.com，核心目标是引导用户加微信进入私域社群。

## 用户故事

**作为一个跨境电商卖家**，我访问 ai.yzsgo.com，看到鸭嘴售这个 AI 跨境电商工具的介绍，觉得功能和案例靠谱，扫码加微信进群，进一步了解和使用产品。

## 品牌定义

- **品牌名**：鸭嘴售（谐音鸭嘴兽，视觉 IP 用鸭嘴兽形象）
- **Slogan**："你的 AI 首席运营官"
- **品牌调性**：活泼有趣，用鸭嘴兽 IP 形象做点缀，不过度卡通化
- **品牌关系**：独立品牌，不提及 Optima AI
- **SEO 策略**：品牌名用"鸭嘴售"，但页面中自然出现"鸭嘴售（谐音鸭嘴兽）"以覆盖"鸭嘴兽 AI"搜索词
- **目标用户**：跨境电商卖家（Amazon、TikTok Shop、Temu、SHEIN 等平台）
- **核心转化**：引导加微信 / 进社群

## 页面结构

单页落地页，7 个板块从上到下：

### 1. Nav 导航栏

- Logo（鸭嘴售）+ 锚点导航（功能 / 案例 / 定价）+ 微信 CTA 按钮
- 滚动时 sticky 固定

### 2. Hero 首屏

- 主标题："你的 AI 首席运营官"
- 副标题："从选品到上架，从客服到投放——告诉 AI 你想卖什么，剩下的交给鸭嘴售"
- 右侧/下方：产品对话界面截图（展示 AI 对话建店的真实场景）
- CTA：微信二维码 + "扫码加入卖家群"
- 鸭嘴兽吉祥物形象点缀（戴耳机/拿包裹，暗示跨境电商+AI）

### 3. 功能展示

5 个核心能力卡片（基于黎阳跨境团队真实痛点提炼）：

| 图标 (Lucide) | 功能 | 描述 |
|------|------|------|
| `Search` | AI 选品分析 | 市场趋势、竞品评论、关键词提取、热卖品监控，告诉你"卖什么能赚" |
| `PackagePlus` | AI 上架 | 采集→AI 生成商品图→多语言 Listing→多店铺发布，晚上跑完早上验收 |
| `MessageSquare` | 达人建联 | 批量邀约、自动回复、样品跟踪，达人合作全流程 AI 搞定 |
| `Video` | AI 视频生成 & 剪辑 | 从商品图一键生成推广视频，或上传素材 AI 自动剪辑成片 |
| `TrendingUp` | 广告投放助手 | 智能建议出价、自动排除低效素材、投放数据一目了然 |

### 4. 使用流程

3 步流程图：
1. 扫码进群
2. 开通账号
3. 对话开店

目的：降低行动门槛，让用户觉得"很简单"。

### 5. 客户案例

2-3 个商家故事（V1 先用占位数据，后续替换真实案例）：
- 案例 1：Temu 铺货卖家，AI 上架效率从每天 15 条提升到 200 条
- 案例 2：Amazon 卖家，AI 评论分析 + 选品建议，找到蓝海品类
- 案例 3：TikTok 卖家，AI 自动回复达人，邀约转化率提升 3 倍

含数据对比，提供社会证明。

### 6. 定价方案

三档对比表，人民币标价：
- **Free**：基础功能，有 AI 额度限制
- **Pro ¥199/月**：完整功能 + 更多 AI 额度
- **Enterprise ¥699/月**：无限 AI + 专属支持 + 定制

### 7. Footer

- 微信二维码（大尺寸）+ "扫码加入卖家群"
- FAQ 区域：
  - "鸭嘴售是什么？" — AI 驱动的跨境电商运营平台
  - "支持哪些平台？" — Amazon、TikTok Shop、Temu、SHEIN 等
  - "需要有电商经验吗？" — 不需要，AI 引导全流程
  - "数据安全吗？" — 不接管广告后台，数据本地处理
- 企业信息 + ICP 备案号预留

### CTA 策略

页面中至少 3 处微信引导：Hero、使用流程后、Footer。形式为二维码图片 + "扫码加入卖家群"。

## 视觉设计

### 配色

- 主色：明亮蓝绿色系（活泼但不幼稚，契合科技+跨境）
- 强调色：亮黄/橙色（CTA 按钮、重点标注）
- 中性色：深灰文字 + 浅灰背景
- 整体明亮干净，避免沉闷企业感

### 鸭嘴兽 IP

- Logo：简洁鸭嘴兽剪影/图标 + "鸭嘴售"中文字标
- Hero 区放稍大的吉祥物形象
- 功能卡片和空状态用小鸭嘴兽表情点缀
- 不过度卡通化，吉祥物是点缀而非主体

### 字体

- 中文：系统字体栈（PingFang SC / Microsoft YaHei）
- 英文/数字：Inter 或 DM Sans
- 标题可考虑有个性的中文字体（CDN 加载）

### 动效

- 滚动渐入（Framer Motion，轻量不花哨）
- Hero 区产品截图轻微浮动/视差效果
- CTA 按钮 hover 微动效

## 部署方案

### DNS 配置（并行推进）

- `www.yzsgo.com` → 阿里云轻量应用服务器（用于 ICP 备案）
- `ai.yzsgo.com` → CNAME `cname.vercel-dns.com`（Vercel，落地页立即上线）

### 备案

- 阿里云购买最便宜的轻量应用服务器（仅用于过备案审核）
- `www.yzsgo.com` 解析到阿里云服务器，提交 ICP 备案
- 备案期间服务器上挂"网站建设中"页面
- 备案周期：通常 5-15 个工作日

### 备案通过后

- `@` 和 `www` DNS 切换指向 Vercel
- `ai.yzsgo.com` 继续保留或 301 重定向到主域

### Vercel 部署

- GitHub 仓库关联 Vercel，main 分支 push 自动部署
- 每个 PR 自动生成预览 URL
- Vercel 自动签发 SSL 证书
